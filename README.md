## 🏗️ System Architecture

<div align="center">

### **6-Agent AI Pipeline**

</div>

| Agent | Function | Safety Feature |
|:------|:---------|:---------------|
| **🤖 1. Intake** | Conversational AI assessment | Guardian mode for minors (13-17) |
| **📊 2. Severity** | Low/Moderate/High classification | <0.7 confidence → Human review |
| **🚨 3. Crisis** | Real-time suicidal ideation detection | **Cannot be overridden** |
| **🏥 4. Coding** | ICD-10/CPT code suggestions | Suggestion only, never diagnosis |
| **🎯 5. Matching** | Therapist/group recommendations | Multi-factor scoring algorithm |
| **📝 6. Reports** | Clinical summaries | Auto-sent to therapists |

## 🚀 Quick Start

### **Prerequisites**

```bash
Node.js >= 18.x
Supabase account
Anthropic API key (Claude Sonnet 4)
```

### **Installation**

```bash
# Clone repository
git clone https://github.com/yourusername/hereforyou-ai.git
cd hereforyou-ai

# Install dependencies
npm install

# Environment setup
cp .env.example .env
# Add your keys to .env

# Run database migrations
npm run db:migrate

# Start dev server
npm run dev
```

### **Environment Variables**

```env
# Supabase
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key

# Anthropic
ANTHROPIC_API_KEY=sk-ant-your-api-key

# Optional
VITE_APP_URL=http://localhost:5173
```

### **Database Setup**

```bash
# Run migrations
npx supabase db push

# Seed sample data (optional)
npm run db:seed

# Enable Row-Level Security policies
npm run db:enable-rls
```

### **Supabase Edge Functions Deployment**

```bash
# Deploy all 6 AI agents
npm run deploy:agents

# Or deploy individually
npx supabase functions deploy intake-agent
npx supabase functions deploy severity-classifier
npx supabase functions deploy crisis-detector
npx supabase functions deploy coding-agent
npx supabase functions deploy matching-agent
npx supabase functions deploy report-generator
```
